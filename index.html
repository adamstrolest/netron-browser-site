Netron Browser

A lightweight WebKitGTK browser for the lowest-end machines you own.

Install Now View on GitHub

Why Netron?

⚡ Designed for 32-bit / 1GB RAM

Runs smoothly on single-core CPUs with 1 GB RAM. No Electron, no Chrome — pure WebKitGTK + Python.

📺 YouTube at 5% CPU

Redirects YouTube to yt-dlp + mpv. Video plays at ~5% CPU instead of pegging your machine.

🔒 Privacy First

Per-site cookie control, private browsing mode, WebGL/WebAudio/media-stream toggles, clear-on-exit.

📂 Tabbed + Session Restore

Full tab management, bookmarks, history, per-site zoom, and session restore after crashes.

🎨 Themes + Custom New Tab

Apply CSS themes, set custom background images on new tab pages, switch User Agents.

📥 Chrome-Style Downloads

Bottom shelf download bar with progress, cancel, and "Open folder" — all in a compact GTK widget.

Installation

Debian / Ubuntu / Raspberry Pi OS and derivatives.

Option 1: APT Repository (recommended)

echo "deb [signed-by=/usr/share/keyrings/netron-browser.gpg] https://adamstrolest.github.io/netron-browser-apt stable main" | sudo tee /etc/apt/sources.list.d/netron-browser.list
sudo apt update && sudo apt install netron-browser

Option 2: Direct .deb download

wget https://github.com/adamstrolest/netron-browser-apt/releases/latest/download/netron-browser_1.7.0_all.deb
sudo dpkg -i netron-browser_1.7.0_all.deb

Option 3: Run from source

git clone https://github.com/adamstrolest/netron-browser-site.git
cd netron-browser-site/src
sudo apt install python3-gi gir1.2-webkit2-4.1 gir1.2-gtk-3.0 gir1.2-pango-1.0 gir1.2-gdkpixbuf-2.0 gir1.2-notify-0.7 yt-dlp mpv
python3 netron.py
